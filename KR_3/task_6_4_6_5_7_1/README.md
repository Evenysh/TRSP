# task_6_4_6_5_7_1

## Описание
В данном подпроекте реализованы задания 6.4, 6.5 и 7.1 по дисциплине «Технологии разработки серверных приложений».

### Задание 6.4
Реализованы:
- маршрут `POST /login` для аутентификации пользователя;
- генерация JWT-токена при успешном входе;
- защищённый маршрут `GET /protected_resource`;
- доступ к защищённому ресурсу только при наличии корректного JWT-токена.

### Задание 6.5
Реализованы:
- маршрут `POST /register` для регистрации нового пользователя;
- хранение пользователей в памяти приложения;
- хеширование паролей;
- проверка логина и пароля при входе;
- ограничение количества запросов:
  - `POST /register` — не более 1 запроса в минуту;
  - `POST /login` — не более 5 запросов в минуту.

### Задание 7.1
Реализована ролевая модель доступа:
- `admin`
- `user`
- `guest`

Добавлена проверка прав доступа к маршрутам в зависимости от роли пользователя.

## Структура проекта
- `main.py` — основной файл приложения;
- `requirements.txt` — зависимости проекта;
- `README.md` — инструкция по запуску и проверке.
