# task_6_1

## Описание
В данном задании реализован маршрут `GET /login` с использованием HTTP Basic Authentication.

При корректном вводе логина и пароля сервер возвращает приветственное сообщение.  
При неверных учетных данных сервер возвращает ошибку `401 Unauthorized`.
