# task_8_1_8_2

## Описание
В данном подпроекте реализованы задания 8.1 и 8.2 по дисциплине «Технологии разработки серверных приложений».

### Задание 8.1
Реализованы:
- подключение к SQLite через `database.py`;
- скрипт создания таблицы `users`;
- маршрут `POST /register` для добавления пользователя в базу данных.

### Задание 8.2
Реализован CRUD для ресурса `Todo` с использованием SQLite.

Поля сущности `Todo`:
- `id`
- `title`
- `description`
- `completed`

## Структура проекта
- `main.py` — основной файл приложения;
- `database.py` — подключение к базе данных SQLite;
- `init_db.py` — создание таблиц `users` и `todos`;
- `requirements.txt` — зависимости проекта;
- `README.md` — инструкция по запуску и проверке;
- `users.db` — файл базы данных SQLite.
