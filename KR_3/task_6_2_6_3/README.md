# task_6_2_6_3

## Описание
В данном подпроекте реализованы задания 6.2 и 6.3 по дисциплине «Технологии разработки серверных приложений».

### Задание 6.2
Реализованы:
- регистрация пользователя через `POST /register`;
- авторизация через `GET /login` с использованием HTTP Basic Authentication;
- хеширование паролей с помощью `bcrypt`;
- проверка пароля по хешу.

### Задание 6.3
Реализованы режимы работы приложения через переменные окружения:
- `DEV`
- `PROD`

В режиме `DEV`:
- маршрут `/docs` защищён через HTTP Basic Authentication;
- маршрут `/openapi.json` защищён через HTTP Basic Authentication;
- маршрут `/redoc` недоступен.

В режиме `PROD`:
- маршруты `/docs`, `/openapi.json`, `/redoc` скрыты и возвращают `404 Not Found`.

## Структура проекта
- `main.py` — основной файл приложения;
- `requirements.txt` — зависимости проекта;
- `.env.example` — пример файла переменных окружения;
- `.gitignore` — исключения для Git;
- `README.md` — инструкция по запуску и проверке.
